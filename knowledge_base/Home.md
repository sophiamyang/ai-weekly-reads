# AI Weekly Reads

This Obsidian vault is the reading cockpit for AI Weekly Reads.

## Weekly Flow

1. Add one-off links to `../inbox/links.txt`.
2. Keep recurring sources in `../config/sources.json`.
3. Run the weekly builder.
4. Review notes in `resources/`.
5. Read generated books in `weekly_books/`.

## Folders

- `resources/`: one durable note per video, podcast episode, or web resource
- `raw_transcripts/`: one transcript-only note per resource
- `weekly_books/`: compiled Kindle reading packets
- `sources/`: generated podcast, channel, and publication hubs
- `people/`: generated speaker and guest hubs
- `topics/`: generated subject hubs that organize the global graph
- `templates/`: note shapes for future generated resources

## Topic Tags

Resource notes use 2-4 controlled `topic/...` tags for their central ideas. Structural metadata such as source, status, priority, language, and note type lives in Properties instead of tags.
